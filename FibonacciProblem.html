Fibonacci I guess

Let \(S_n\) be a Fibonacci-like sequence with start numbers \(a\) and \(b)\.

\(S_1 = a\) and \(S_2 = b\)

\(S_3 = a + b\)

\(S_4 = S_2 + S_3 = b + a + b = a + 2b\)

\(S_n = aF_{n-2} + bF_{n-1}\)

Let \(a = F_{k1}\) and \(b = F_{k+1}\).

\(S_n = F_{n-2}F_{k}+F_{n-1}F_{k+1}\)

\(= F_{n-1+k}\)

\(\Rightarrow\)
